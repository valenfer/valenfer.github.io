Coolors
Esta página web es una herramienta para explorar paletas de colores hermosas y accesibles para tu sitio web basadas en las Pautas de Accesibilidad al Contenido Web (WCAG). Puedes crear la paleta perfecta o inspirarte en miles de esquemas de colores hermosos. También puedes obtener información útil sobre el color, como la conversión, las combinaciones, la simulación de la ceguera, el contraste y las bibliotecas de colores. Además, puedes usar Coolors con tus herramientas favoritas, como iOS, Android, Figma, Chrome y Adobe.

Transforms
Es un generador de CSS Transform que te ayuda a generar rápidamente declaraciones de transformación de CSS para tu sitio web. Incluye transformaciones de Rotar, Escalar, Sesgar y Traducir. Viene con muchas opciones y demuestra instantáneamente. La propiedad CSS transform te permite modificar el espacio de coordenadas del modelo de formato visual CSS. Usándola, los elementos pueden ser trasladados, rotados, escalados o sesgados de acuerdo a los valores establecidos. Es una herramienta útil para los desarrolladores web para crear efectos visuales interesantes en sus sitios web.

Regex
Regex101 es una consola interactiva de expresiones regulares que te permite depurar tus expresiones en línea. Esto significa que puedes crear tus expresiones y ver cómo afectan a un conjunto de datos en vivo, todo en una pantalla al mismo tiempo. Es una herramienta muy útil para aprender y dominar las expresiones regulares

UrbanFonts
UrbanFonts es un Recurso en línea donde puedes descargar fuentes gratuitas. Ofrece más de 30,000 fuentes de PC y Mac. Puedes personalizar tu experiencia con vistas previas de fuentes en vivo. Todas las fuentes están categorizadas y pueden ser guardadas para referencia rápida y comparación.

Unsplash
En Unsplash, puedes explorar y descargar imágenes y fotos gratuitas de alta calidad. Animista
En la página Animista, puedes encontrar animaciones básicas que puedes personalizar según tus necesidades. Puedes ajustar varios parámetros de la animación, como la duración, el retraso, la cantidad de veces que se repite, la dirección, el modo de llenado, y más. Una vez que hayas configurado tu animación, la herramienta generará el código CSS correspondiente, que puedes copiar y pegar en tu propio proyecto1. Es una herramienta muy útil para los desarrolladores web, ya que simplifica el proceso de creación de animaciones complejas con CSS.

ParkUI
Park UI es una biblioteca de componentes diseñados de manera hermosa que se construyen sobre Ark UI. Estos componentes funcionan para los frameworks de JavaScript y CSS de tu elección. En GitHub, puedes encontrar más detalles sobre cómo usar Park UI en tus proyectos1. Además, existe un kit de diseño de sistema Park UI en Figma, que incluye una colección de componentes reutilizables que puedes copiar y pegar sin esfuerz en tus aplicaciones.
