CoolorsEsta página web es una herramienta para explorar paletas de colores hermosas y accesibles para tu sitio web basadas en las Pautas de Accesibilidad al Contenido Web (WCAG). Puedes crear la paleta perfecta o inspirarte en miles de esquemas de colores hermosos. También puedes obtener información útil sobre el color, como la conversión, las combinaciones, la simulación de la ceguera, el contraste y las bibliotecas de colores. Además, puedes usar Coolors con tus herramientas favoritas, como iOS, Android, Figma, Chrome y Adobe.

TransformsEs un generador de CSS Transform que te ayuda a generar rápidamente declaraciones de transformación de CSS para tu sitio web1. Usándola, los elementos pueden ser trasladados, rotados, escalados o sesgados de acuerdo a los valores establecidos2. Es una herramienta útil para los desarrolladores web para crear efectos visuales interesantes en sus sitios web.
